# 张祥前 & 徐玉川 合作论文库 (21 份)
# Research Archive: Zhang Xiangqian & Xu Yuchuan (21 Files)

本仓库包含了张祥前与徐玉川合作撰写的 21 份研究文件。这些文献记录了双方在《统一场论》理论推导及相关物理实验方面的共同研究成果。

This repository contains 21 research documents co-authored by **Zhang Xiangqian** and **Xu Yuchuan**. These files document their collaborative work on **Unified Field Theory** and experimental physics.

---

## 📄 主要研究内容 (Core Research Topics)

* **理论推导 (Theoretical Derivations)**: 包含关于统一场论核心逻辑与数学框架的系统论述。
* **动量公式分析 (Momentum Analysis)**: 详细探讨了统一场论动量公式 $P = m(C - V)$ 及其与经典力学动量的关系。
* **实验验证 (Experimental Verification)**: 关于利用变化电磁场与非晶纳米材料产生引力效应的观测与分析。
* **五大常数关联 (Five Constants Correlation)**: 探讨物理常数 $G, c, h, \nu, r, T$ 在几何空间模型下的归一化与统一。
